 Project : Neighbourhood Map
=============================

## How to Run

-> Go to main folder.
-> Run index.html.
-> Click on markers to check rating and likes.

## COncept Used

-> Google Maps Api
	* To load the map.
-> Knockout js
	* To Bind the data.
-> foursquare id
	* To fetch the info.
-> Bootstrap
	* For styling.
-> Jquerry
	* TO add functionality.
